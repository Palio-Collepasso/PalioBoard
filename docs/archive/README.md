# Archive

This directory stores historical material that may still help with migration, traceability, or examples, but is **not** part of the default reading path for implementation work.

## Files and subdirectories
- `product qna.md` — archived product deep Q&A that has been superseded by authoritative docs or condensed active Q&A
- `architecture-qna.md` — archived architecture deep Q&A that has been superseded by authoritative docs or condensed active Q&A
- `milestones.md` — long document with all the milestones
- `palio/` — real event description and regulation

## Rule
Do not treat archived material as authoritative unless an active doc explicitly points back here for a specific task.
