# Q&A

This directory is a small temporary clarification layer.

## Active files
- `open-questions.md` — unresolved questions only
- `recently-resolved.md` — short-lived promotion map for answers that were already moved into authoritative docs

## Rules
- Do not use this directory as a second architecture or product spec.
- When an answer becomes stable, move it into the authoritative document and remove it from active Q&A.
- Keep only unresolved or recently resolved items here.
- Older deep Q&A lives in `docs/archive/qna/` and is not part of the default reading path.
