# Apps

This directory groups the runnable application roots for the monorepo.

- `apps/api/` is reserved for the FastAPI backend and its tests
- `apps/web/` now contains the Angular SPA scaffold, lazy shell routing, and frontend boundary checks

The application subtrees are still mid-bootstrap: the frontend shell scaffold exists, while backend, infra, and the fuller test harnesses continue to land through the `m-0` foundation work.
